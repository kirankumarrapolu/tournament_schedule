# tournament_schedule

Simple 3-team badminton round-robin scheduler (best of 3 games per match).

## Usage

1. Run:

```bash
python3 tournament.py
```

2. Enter team names one per line. Blank line when done (minimum 2 teams required).
3. The program auto-generates a round-robin schedule (every team plays every other team).
4. For each match, enter the game winner for best-of-3.
5. After input, you get match-by-match results + final standings and champion.

### Quick example input

- Ashok and Manisha
- Chari and Kiran
- Manoj and Rajan
- <blank>

Then match results flow begins.
